# JULIEN_

Julien é um assistente de IA com interacao somente por voz, criado por CauaSarraf como projeto de estudos sobre LLMs e aplicacoes praticas de inteligencia artificial.

O foco do projeto e explorar integracao, experimentacao e aprendizado com modelos de linguagem, utilizando Ollama como base para execucao local dos modelos.

Objetivo inicial: entregar uma experiencia no estilo Jarvis, em que o usuario fala com o Julien e recebe respostas faladas, sem fluxo de chat por texto como interface principal.
